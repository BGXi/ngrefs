# `ng get`

> Удобная функция для получения значения из конфига для автоматизации или работы с глобальными конфигами.

```
ng get [key]
```

Получение настройки из `.angular-cli.json`.

## `--global`

* Тип: `Boolean`
* Псевдонимы: `-g`
* По-умолчанию: `false`

Получение настройки из глобальный конфиг Angular CLI.
