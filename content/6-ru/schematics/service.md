# `service`

ID: `@schematics/angular:service`

Generates a service.

```
ng generate service [name]
```


## `project`

* **Тип**: `string`

Имя проекта.


## `spec`

* **Тип**: `boolean`
* **По-умолчанию**: `true`

Specifies if a spec file is generated.


## `flat`

* **Тип**: `boolean`
* **По-умолчанию**: `true`

Flag to indicate if a dir is created.
