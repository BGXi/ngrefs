# `class`

ID: `@schematics/angular:class`

Генерация класса.

```
ng generate class [name]
ng g class [name]
```

## Параметры

### `project`

* **Тип**: `string`

Имя проекта.


### `type`

* **Тип**: `string`

Specifies the type of class.


### `spec`

* **Тип**: `boolean`
* **По-умолчанию**: `false`

Генерация файла тестов.
