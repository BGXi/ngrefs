# `guard`

ID: `@schematics/angular:guard`

Generates a guard.

```
ng generate guard [name]
```


## `project`

* **Тип**: `string`

Имя проекта.


## `spec`

* **Тип**: `boolean`
* **По-умолчанию**: `true`

Specifies if a spec file is generated.


## `flat`

* **Тип**: `boolean`
* **По-умолчанию**: `true`

Flag to indicate if a dir is created.


## `module`

* **Псевдоним**: `m`
* **Тип**: `string`

Allows specification of the declaring module.
